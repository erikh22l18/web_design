Durante la sesión de React

Se explicaron los conceptos fundamentales de React, como el Virtual DOM (Documento de Objeto Modelo Virtual) y el enlace unidireccional de datos. El Virtual DOM es una representación en memoria de la estructura de la interfaz de usuario, que permite a React optimizar y actualizar eficientemente los cambios en la misma.

Además, se abordó el concepto de JSX, que es una extensión de sintaxis de JavaScript utilizada en React para definir la estructura de los componentes. JSX combina HTML y JavaScript en un solo archivo, lo que facilita la creación de interfaces de usuario dinámicas.

Durante la sesión práctica, los participantes aprendieron a configurar un proyecto de React utilizando herramientas como Create React App. También se les enseñó cómo crear componentes simples y comprender la estructura de un componente de React, incluyendo la función de renderizado y el manejo de propiedades (props).

Además, se exploraron conceptos esenciales de React, como el estado (state) y los ciclos de vida (lifecycle) de los componentes. El estado permite que los componentes React mantengan y actualicen datos internos, mientras que los ciclos de vida ofrecen métodos que se ejecutan en diferentes etapas del ciclo de vida del componente, como cuando se monta o se desmonta.
